# PlaneGamePro-For-Java
### 简介

使用Java图形化开发的飞机大战游戏高级版

### 特点

- 可以自定义敌方单位数量
- 可以自定义难度
- 可以自定义boss生命值
- 可以自定义飞行速度
- 添加了量化生命值功能
- 添加了胜利和失败结算动画
